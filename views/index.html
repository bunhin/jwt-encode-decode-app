JSON Web Token (JWT) Encode / Decode App

This is a simple app that allows you to encode and decode JWTs and see the results.

  • Encode a JWT
  • Decode a JWT

To create your private and public keys, run these openssl commands:

  • openssl genrsa -out private.pem 1024
  • openssl rsa -in private.pem -pubout > public.pem
